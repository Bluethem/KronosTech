# Requerimientos no Funcionales

## RNF-001: Performance

| **ID**    | **Descripción**                              | **Métrica**   |
| --------- | -------------------------------------------- | ------------- |
| RNF-001.1 | Tiempo de respuesta de la API (95 percentil) | < 200 ms      |
| RNF-001.2 | Tiempo de carga de la página principal       | < 2 s         |
| RNF-001.3 | Soporte para usuarios concurrentes           | 1000 usuarios |
| RNF-001.4 | Tiempo total del proceso de checkout         | < 30 s        |

## RNF-002: Seguridad

| **ID**    | **Descripción**                                       |
| --------- | ----------------------------------------------------- |
| RNF-002.1 | Las contraseñas deben hashearse con Argon2 o bcrypt   |
| RNF-002.2 | Toda la comunicación debe usar HTTPS                  |
| RNF-002.3 | Tokenización de datos de tarjetas (PCI-DSS)           |
| RNF-002.4 | Implementar rate limiting (100 req/min por IP)        |
| RNF-002.5 | Uso de JWT con expiración de 1 hora                   |
| RNF-002.6 | Registro de logs de acciones críticas (ventas, pagos) |
| RNF-002.7 | Protección contra SQL Injection                       |
| RNF-002.8 | Protección contra XSS                                 |
